# :saxophone: ColtrAIne (WIP)
A soloist LSTM model that learns to improvise jazz, so I don't have to. 

Trained with solos from the [Weimar Jazz Database](https://jazzomat.hfm-weimar.de/dbformat/dboverview.html).
